# 🏋️ Fitness Dashboard

A modern, responsive Fitness Dashboard built using HTML, CSS, JavaScript, and Chart.js. This web application helps users monitor their daily fitness activities, calculate BMI, track water intake, monitor sleep, and visualize progress through interactive charts.

## ✨ Features
- 📊 Interactive Dashboard
- 👣 Daily Step Tracker
- 🔥 Calories Burned Calculator
- ⚖️ BMI Calculator
- 💧 Water Intake Tracker
- 🛌 Sleep Tracker
- ❤️ Heart Rate Display
- 📅 Workout Planner
- 📈 Weekly Progress Charts
- 🌙 Dark & Light Mode
- 💾 Local Storage Support
- 📱 Fully Responsive Design

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Local Storage

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/your-username/Fitness-Dashboard.git
```

2. Navigate to the project folder
```bash
cd Fitness-Dashboard
```

3. Open `index.html` in your browser.
No additional installation is required.
